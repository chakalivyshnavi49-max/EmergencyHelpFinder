# 🚨 Emergency Help Finder

A smart web application that helps users quickly find nearby emergency services like hospitals, police stations, and fire stations based on their situation.

---

## 📌 Project Description

This project is designed to assist users during emergencies by:
- Detecting the type of emergency
- Getting user location
- Showing nearby services
- Sending SOS alerts
- Allowing service providers to view emergency requests

---

## 🚀 Features

- 📍 Get current location
- 🔍 Detect emergency type (accident, fire, crime)
- 🗺️ View nearby help in Google Maps
- 🎤 Voice input support
- 🆘 SOS emergency alert system
- 👥 Role-based login (Hospital, Police, Fire, Helper)
- 💾 Local storage-based emergency data system
